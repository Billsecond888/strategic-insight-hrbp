# 战略洞察与组织破局

> 基于毛泽东思想方法论的 HRBP 高阶技能——将实事求是、群众路线、独立自主三大灵魂转化为专业能力。

## 概述

本技能融合了毛泽东思想的三大核心方法论与麦肯锡结构化思维，形成一套「有高度、能落地、讲得通」的本土化 HRBP 战略方法论。

### 核心逻辑映射

| 毛泽东思想灵魂 | HRBP 专业能力 | 核心原则 |
|---------------|--------------|---------|
| **实事求是** | 深度组织诊断 | 拒绝照搬模型，基于业务真相 |
| **群众路线** | 利益相关者管理与文化落地 | 从员工中来，到员工中去 |
| **独立自主** | 定制化解决方案 | 不迷信最佳实践，解决自家问题 |

## 四大技能模块

1. **调查研究力（实事求是）** —— 蹲点调研、阶级分析法、业务现场真实痛点清单
2. **矛盾分析法（抓主要矛盾）** —— 主要矛盾识别、矛盾转化策略、季度关键战役路线图
3. **群众工作法（群众路线）** —— 试点先行（建立根据地）、思想建党（文化共识）
4. **战略持久战（论持久战）** —— 防御/相持/反攻三阶段、游击战术（微创新积累）

## 话术示例

**Before（传统 HR 话术）**：「员工满意度下降，建议优化薪酬体系。」

**After（毛选风格）**：「经过对一线 50 名员工的深度访谈（调查研究），我们发现当前主要矛盾并非薪酬绝对值偏低，而是分配机制不公导致的核心骨干流失（抓主要矛盾）。建议先在研发部试点新的积分激励制（建立根据地），验证效果后再全员推广（星星之火可以燎原），预计通过 6 个月的持久战实现人才结构优化。」
