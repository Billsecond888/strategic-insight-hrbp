# Step 3：群众动员（群众路线）

## 目标
干系人分析 + 试点策略设计。

## 核心原则
- 从群众中来，到群众中去
- 先试点、再推广
- 团结一切可以团结的力量

## 分析步骤

### Step 3.1：干系人地图
识别组织中的三股势力：
- 🟢 进步势力（~20%）：变革支持者
- 🟡 中间势力（~60%）：观望者
- 🔴 顽固势力（~20%）：变革阻力

### Step 3.2：试点选择
| 标准 | 权重 | 评分(1-5) |
|------|:----:|:---------:|
| 配合度 | ⭐⭐⭐⭐⭐ | |
| 代表性 | ⭐⭐⭐⭐ | |

| 规模 | ⭐⭐⭐ | |
| 影响力 | ⭐⭐⭐⭐⭐ | |

### Step 3.3：动员策略设计
针对不同势力制定差异化策略。
