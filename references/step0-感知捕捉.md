# Step 0：感知捕捉

## 目标
把模糊感觉/素材/任务变成可分析的输入。

## 教练式追问（3 轮）

### 第 1 轮：打开话题
- "你最近在关注什么业务问题？"
- "是什么让你觉得'哪里不对'？"
- "如果用一个比喻形容现在的状态，你会用什么？"

### 第 2 轮：聚焦范围
- "这个问题主要影响哪个团队/部门？"
- "你手头有什么数据或素材？"
- "这个问题存在多久了？"

### 第 3 轮：明确期望
- "你希望这次分析产出什么？"
- "谁需要看到这个产出？"
- "什么时候需要？"

## 输出格式

```
## 初步问题陈述
[一句话描述]

## 问题类型
[ ] 诊断型 — "到底发生了什么？"
[ ] 处方型 — "我们该怎么做？"
[ ] 影响型 — "这会带来什么后果？"

## 已有素材清单
- [素材 1]
- [素材 2]

## 缺失信息
- [信息 1]
- [信息 2]
```
