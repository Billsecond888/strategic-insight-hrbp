# 模块四：战略持久战（论持久战）

> **核心原则**：组织变革没有速胜，只有持久战。分阶段、设里程碑、积小胜为大胜。

## 1. 核心理念

毛泽东在《论持久战》中批判了"亡国论"和"速胜论"，提出抗日战争将经历三个阶段：战略防御、战略相持、战略反攻。

**HRBP 应用**：组织变革同样没有捷径。不要期望三个月改变文化，也不要因为短期看不到效果就放弃。分阶段推进，管理好预期。

## 2. 三阶段划分

### 2.1 阶段总览

```
阶段一：战略防御（稳住基本盘）
  ├─ 目标：止血、维稳、建立信任
  ├─ 时长：1-3 个月
  └─ 关键动作：诊断、沟通、小范围试点

阶段二：战略相持（逐步渗透）
  ├─ 目标：扩大影响、积累胜势
  ├─ 时长：3-6 个月
  └─ 关键动作：推广、固化、培养骨干

阶段三：战略反攻（全面见效）
  ├─ 目标：规模化、制度化、文化内化
  ├─ 时长：6-12 个月
  └─ 关键动作：全面推行、制度固化、持续优化
```

### 2.2 各阶段详细设计

#### 阶段一：战略防御（1-3 个月）

| 维度 | 内容 |
|------|------|
| **核心目标** | 稳住基本盘，不出现大的动荡 |
| **关键动作** | ① 全面调研诊断（调查研究力）<br>② 识别主要矛盾（矛盾分析法）<br>③ 建立信任关系（群众工作法）<br>④ 选择 1-2 个试点（建立根据地） |
| **资源投入** | 调研 30% + 沟通 40% + 试点 30% |
| **胜利标准** | ① 完成调研报告<br>② 主要矛盾得到管理层共识<br>③ 试点团队启动运行 |
| **风险预警** | 如果出现大规模离职或公开对抗，延长防御期 |

#### 阶段二：战略相持（3-6 个月）

| 维度 | 内容 |
|------|------|
| **核心目标** | 扩大试点成果，逐步渗透 |
| **关键动作** | ① 试点复盘迭代<br>② 培养变革骨干（进步势力）<br>③ 争取中间势力<br>④ 扩大试点范围 |
| **资源投入** | 试点推广 40% + 培训赋能 30% + 沟通 30% |
| **胜利标准** | ① 试点验证成功<br>② 培养出 3-5 名变革骨干<br>③ 覆盖 30% 以上团队 |
| **风险预警** | 如果试点效果不达预期，退回防御期调整 |

#### 阶段三：战略反攻（6-12 个月）

| 维度 | 内容 |
|------|------|
| **核心目标** | 全面推行，制度固化，文化内化 |
| **关键动作** | ① 全面推行变革方案<br>② 将成功经验写入制度流程<br>③ 建立持续改进机制<br>④ 庆祝胜利，树立标杆 |
| **资源投入** | 全面推行 50% + 制度固化 30% + 文化建设 20% |
| **胜利标准** | ① 覆盖 90% 以上团队<br>② 关键指标达到预期<br>③ 变革成为"我们这里做事的方式" |
| **风险预警** | 如果出现反弹，启动"回头看"机制 |

## 3. 游击战术（微创新积累）

### 3.1 什么是游击战术

在资源不足时，利用灵活的小型项目积累胜势，避免与大部门正面硬刚。

### 3.2 游击战术原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **敌进我退** | 阻力大时暂避锋芒 | 不强行推行，先做能做的事 |
| **敌驻我扰** | 对方松懈时持续影响 | 定期推送小案例、小数据 |
| **敌疲我打** | 对方疲惫时集中突破 | 在对方最需要支持时提供帮助 |
| **敌退我追** | 对方退让时扩大战果 | 抓住窗口期快速推进 |

### 3.3 微创新清单

| 类型 | 示例 | 投入 | 见效周期 |
|------|------|:----:|:--------:|
| 流程优化 | 简化审批流程 | 低 | 1-2 周 |
| 工具改进 | 引入协作工具 | 中 | 2-4 周 |
| 沟通创新 | 建立快速反馈群 | 低 | 即时 |
| 培训微调 | 增加案例分享环节 | 低 | 1 周 |
| 激励小改 | 增设即时认可机制 | 低 | 2 周 |

## 5. 预期管理话术

| 场景 | 话术 |
|------|------|
| 老板问"什么时候能看到效果" | "变革就像种树，前三个月是扎根期，表面看不到变化但根系在生长。我建议我们设定三个里程碑，第一个里程碑在第 X 周，届时可以看到初步效果。" |
| 团队说"怎么还没变化" | "我们现在处于战略防御阶段，核心任务是稳住基本盘、打好基础。就像打仗一样，先站稳脚跟再出击。预计第 X 周进入相持阶段后，大家会明显感受到变化。" |
| 自己感到焦虑 | "持久战最怕的不是慢，而是乱。只要方向对、节奏稳，积小胜为大胜，最终一定能实现目标。回头看，每一步都算数。" |

## 6. 常见误区

| 误区 | 表现 | 正确做法 |
|------|------|---------|
| 速胜论 | 期望 3 个月改变文化 | 接受至少 12-18 个月的变革周期 |
| 亡国论 | 遇到阻力就放弃 | 区分"暂时困难"和"根本不可行" |
| 阶段跳跃 | 防御期没站稳就进入反攻 | 每个阶段必须达到胜利标准才能进入下一阶段 |
| 里程碑模糊 | "有明显改善"无法衡量 | 每个里程碑必须有可验证的完成标志 |
| 忽视庆祝 | 只赶路不庆祝 | 每个里程碑达成后必须庆祝，保持士气 |
